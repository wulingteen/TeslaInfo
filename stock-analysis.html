Coverage Notes

給車主看的研究筆記

這裡不做買賣建議，只整理如果車主想了解 TSLA 和相關企業，該看哪些可交易標的和常見風險。

TSLA 核心敘事股

估值不是在買汽車公司，而是在買多條選擇權

Base case 是汽車 + Energy 撐住現金流；bull case 是 FSD/Robotaxi/Optimus 開始被市場重新定價； bear case 是交付成長、毛利與監管風險同時壓縮估值倍數。

  • 催化劑：季度交付、毛利率、Energy 毛利、FSD 里程碑。
  • 風險：價格戰、需求彈性、Robotaxi 時程、管理層注意力。
RKLB / ASTS / IRDM / GSAT SpaceX / Starlink 代理籃子

SpaceX 未上市，太空籃子的作用是讀市場風險偏好

RKLB 和 ASTS 偏高 beta 成長，IRDM 偏成熟通訊現金流，GSAT 偏頻譜與裝置合作選擇權。 這些不是 SpaceX 替代品，而是觀察太空資本市場情緒的代理。

  • 催化劑：發射成功率、衛星部署、政府合約、直連手機進度。
  • 風險：資本支出、監管頻譜、技術延遲、競爭補貼。
NVDA / GOOGL / MSFT / META xAI / X 對照籃子

xAI 的可交易讀法，是算力、平台與分發效率

NVDA 反映 AI capex，GOOGL/MSFT 反映模型與雲端平台，META 反映社群平台如何把 AI 變成廣告效率。 xAI 和 X 未上市，所以應該用這些公司建立相對估值座標。

  • 催化劑：模型版本、API 採用、企業合約、X Premium+ 留存。
  • 風險：算力成本、開源競爭、平台信任、監管與內容治理。
RIVN / LCID EV 同業壓力測試

EV 同業能幫 TSLA 讀出需求與資本市場風向

Rivian 比較像中價位與商用車成長參考；Lucid 更像高端 EV 需求與資本需求壓力參考。 如果同業同時承壓，TSLA 的汽車端估值溢價會更依賴軟體與能源驗證。

  • 催化劑：交付量、單車成本、現金消耗、車型上市。
  • 風險：需求放緩、資金稀釋、價格戰、供應鏈壓力。
